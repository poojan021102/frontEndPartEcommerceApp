# ecommerce-app

UI for E-Commerce-App using React.js and Bootstrape styling:

## Features:
-> Authentication using Localstorage
-> Shopping Cart 
-> Wishlist
-> Stripe Payment Gateway Checkout

## Technologies Used:
-> React.js
-> Redux
-> Bootstrape css
-> Node.js & Express.js for payment gateway integration
